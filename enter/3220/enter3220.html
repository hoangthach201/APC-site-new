APC Enter 3.2.2.0

Changelog APC Enter 3.2.2.0 - nhật ký thay đổi

- Thêm hàm tạo mã QR để chèn vào hồ sơ =qrcode(Gia_tri)
- Thêm hàm đọc số bằng tiếng Anh = docsoen(Gia_tri)
- Sửa lỗi khi gặp thông báo “Thời gian hệ thống bị thay đổi”: thường xuyên báo hết hạn nhập mã phản hồi
- Khách hàng\Dữ liệu khách hàng\Chuyển KH sang user khác: sửa lại sắp xếp hiển thị danh sách user xổ xuống theo vần ABC
- Thông tin\Thông tin chung\Nhập Excel: sửa lỗi đôi lúc nhập file excel xong thoát mở lại chương trình APC thì báo lỗi “Database is locked”
- Tiện ích: Thêm màn hình “Tạo mã QR” để tạo mã QR từ thông tin do người dùng nhập vào
- Tiện ích\Tính lãi: Thêm chức năng tự động phân cách phần nghìn khi nhập vào số tiền
- Tiện ích\Tính lãi: Bổ sung thêm tùy chọn "Tính lãi cả ngày đầu và ngày cuối"
- Cấu hình\Hồ sơ, form Sửa: Sửa lỗi khi thay đổi Module nhưng danh sách nhóm hồ sơ xổ xuống tương ứng không thay đổi theo
- Cấu hình\Sửa thuộc tính file mẫu: Cho phép tìm kiếm và thay thế đối với file docm
- Form hồ sơ\Danh sách: chỉnh lại khi bấm X (đóng form danh sách) thì tương đương với bấm "Xong"
- Clipboard: Lỗi khi dữ liệu có dấu chấm thập phân ví dụ ".000" ở clipboard khi nhập vào APC bị hiểu thành phân cách phần nghìn
- Form hồ sơ\nút Scan QR\nút Chọn ảnh hoặc file pdf: sửa lỗi không đọc được mã QR trong một số ảnh chụp
- Form hồ sơ\Scan QR\nút "Chọn ảnh hoặc file pdf": Chỉnh lại sau khi đọc ra kết quả thì tự động load vào combobox đầu tiên
- Form hồ sơ\nút "Nhập file bk" và Khách hàng\Nhập file dữ liệu khách hàng: nhập file bk: sửa lỗi tên tab trong danh sách bị reset về mặc định
- Sửa lỗi một số lỗi liên quan đến thiết lập định dạng vùng, định dạng
