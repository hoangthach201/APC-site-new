Hướng dẫn copy APC từ máy tính này sang máy tính khác

1. Nếu bạn biết CHÍNH XÁC thư mục APC của bạn nằm ở đâu thì chỉ việc vào đó copy CẢ THƯ MỤC APC.
2. Nếu APC của bạn đang ở dạng Shortcut trên màn hình nền hoặc trên thanh Start bar thì thực hiện theo hướng dẫn dưới đây:
- Trường hợp file chạy APC là shortcut ở màn hình nền: Bấm chuột phải vào biểu tượng APC chọn “Open file location”:

-> ở hộp thoại xuất hiện, bấm chọn “Open File Location”:
