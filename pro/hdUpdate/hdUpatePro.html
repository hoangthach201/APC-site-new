Hướng dẫn cập nhật phiên bản mới cho APC Pro

Áp dụng cho APC Pro đang từ phiên bản 3.5.5 trở lên

Xem phiên bản APC hiện tại của bạn ở góc trái bên dưới gần chỗ hiển thị ngày sử dụng còn lại:
(Như ảnh minh họa dưới đây là APC Pro phiên bản 5.9.0)

Bước 1: Tải file cập nhật phiên bản về máy

Bạn có thể tải file cập nhật phiên bản mới nhất tại đây: Tải về

File cập nhật phiên bản là file nén có tên Update_Pro_x.y.z.zip (x.y.z là ký hiệu phiên bản, ví dụ: Update_Pro_6.0.0.zip)
KHÔNG giải nén file này!
File này cần để ở ổ D hoặc E... trong máy (KHÔNG được để ở ổ C hoặc ổ mạng sẽ dễ bị lỗi, nếu máy chỉ có ổ C thì có thể để ở Desktop).

Bước 2: Thực hiện cập nhật phiên bản

Vào menu Hệ thống \ Cập nhật phiên bản mới:

Bấm "Chọn file" rồi chọn đến file Update_Pro_xyz đã tải, sau đó bấm Open:

Bấm Cập nhật để tiến hành:

Chờ vài phút để chương trình tự động cập nhật, trong quá trình này KHÔNG tắt chương trình hoặc tắt máy tính!
Bấm OK chương trình sẽ tự mở lại:

Xong có thể sử dụng bình thường!
